# AI Study Companion

A web-based application that helps students improve productivity by connecting study tracking, AI guidance, and progress analysis in one system.

## Features
- Study timer with subject tracking to monitor learning sessions  
- AI-based assistant that gives guidance based on study activity  
- Progress analysis with summaries and improvement suggestions  
- Notes system with highlight feature for important points  
- Automatic PDF generation from highlighted notes  
- Text-to-handwriting conversion for better note presentation  

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
- Chart.js

## Description
This project focuses on improving student productivity by integrating study tracking, AI interaction, and progress monitoring. The system uses study session data to guide users, provide insights, and help manage notes effectively in a single platform.
